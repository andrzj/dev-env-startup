# Development Environment Setup Script

A script to install some development tools in Linux (as today in Ubuntu 16.04 LTS)

## Getting Started

```shell
./dev_env_startup.sh
```

The script will install to you:

- Git
- Node + NPM
- Mongo
- MySQL
- Docker

That's it. So far.
